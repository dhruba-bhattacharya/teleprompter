# Teleprompter Studio

A browser-based teleprompter with:

- pasted text input
- local PDF / DOC / DOCX / TXT import
- remote PDF / Word URL loading
- auto-scroll playback
- microphone-driven voice follow
- adjustable visible window, text size, spacing, width, mirror mode, and all caps

## Run locally

```bash
npm start
```

Then open `http://localhost:4173`.

## Other commands

```bash
npm run build
npm run preview
```
